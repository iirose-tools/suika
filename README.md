# Suika
## 食用方法
- 把 `config.example.ts` 复制并改名为 `config.ts`
- 修改 `config.ts`
- 安装依赖, 运行 `npm install`
- 构建项目, 运行 `npm run build`
- 启动项目, 运行 `node ./dist/app.js` ( 推荐使用 pm2 等工具启动本BOT )
